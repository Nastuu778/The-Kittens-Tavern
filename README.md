**Описание проекта**

The Kittens Tavern — это интерактивная однопользовательская приключенческая игра, разработанная на языке Python с использованием фреймворка Pygame. Игра представляет собой учебный проект, демонстрирующий навыки объектно-ориентированного программирования, работы с графикой и построения игрового цикла.

Игрок управляет персонажем (котёнком), который исследует три локации: деревню, лес и озеро. В процессе игры необходимо выполнить три квеста типа «найди-принеси»: собрать целебные травы для садовода, доставить зелье старичку и найти потерянную карту торговца. После выполнения всех заданий игрок получает награду.

**В проекте реализованы следующие механики:**

1. Система перемещения персонажа с поддержкой ускоренного бега и проверки коллизий
2. Система диалогов с ветвлением в зависимости от прогресса квестов
3. Квестовая система с отслеживанием прогресса и наград
4. Инвентарь с поддержкой стеков предметов
5. Система камер с масштабированием и следованием за игроком
6. Загрузка карт из формата Tiled (.tmj)

**Требования**

Для запуска проекта необходимо:

1. Python версии 3.8 или выше
2. Библиотека Pygame версии 2.1 или выше

**Установка и запуск**

1. Откройте терминал или командную строку в директории проекта.
2. Создайте виртуальное окружение:

python -m venv venv

3. Активируйте виртуальное окружение:

--Для Windows:

.\venv\Scripts\activate

--Для Linux и macOS:

source venv/bin/activate

4. Установите необходимые зависимости:

pip install -r requirements.txt

5. Запустите игру:

python src/main.py

**Управление**

1. W, A, S, D или стрелки — перемещение персонажа по локации
2. Shift (удерживать) — ускоренный бег
3. E — взаимодействие с объектами (начало диалога с NPC, подбор предмета)
4. Q — подтверждение подбора предмета
5. R — отмена подбора предмета
6. Пробел — переход к следующей реплике в диалоге
7. B — открытие и закрытие инвентаря
8. Esc — закрытие текущего окна (диалог, инвентарь, меню подбора)
